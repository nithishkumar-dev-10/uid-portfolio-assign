## 📂 Section Details

### 🔹 Navbar
- Sticky navigation bar  
- Smooth scrolling between sections  
- Provides quick navigation across the website  

### 🔹 Hero Section
- Displays personal introduction  
- Includes:
  - Name  
  - Role  
  - Short description  
- Contains social links (GitHub, LinkedIn)  

### 🔹 Skills Section
- Built using Flexbox  
- Organized into category-based cards:
  - Programming  
  - Machine Learning  
  - Backend  
  - Tools  
- Skills displayed as pill-style UI components  

### 🔹 Projects Section
- Built using CSS Grid  
- Each project card contains:
  - Title  
  - Description  
  - Tech stack  
  - GitHub link  

Projects included:
- Kitchen Stock AI  
- TransReliant  
- TrackIt Expense Tracker  

### 🔹 Achievements Section
- Grid-based layout  
- Displays certificates and recognitions  
- Each card includes:
  - Image  
  - Title  
  - Description  

### 🔹 Contact Section
- Provides quick contact options:
  - Email  
  - GitHub  
  - LinkedIn
